# Ozon

Инженерная студия для моделирования и визуализации непрерывной озонной очистки воды. Интерфейс вдохновлён операторскими панелями и поддерживает добавление модулей процесса.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

После этого перейдите на `http://localhost:8000`.
